YOLO: please install Yolov5 as instructed in: https://github.com/ultralytics/yolov5
move the following files into the yolov5 folder: detect.py(replace original), display.py, slice_os.py, stitch.py, yolo_detect.py, overlap.py, ./runs/train/weights/best.pt
Inference: run yolo_detect.py --> possible arguments: 
    --source(path to images) 
    --weights(path to weights) 
    --conf-thres(minimum confidence level for detection) 
    --project(save output to project/name) 
    --name(save output to project/name)
Training:..

Faster-RCNN:  get detect_frcnn.zip and unzip it
plesae follow these instructions to install faster-RCNN into the same folder as Faster-RCNN
https://github.com/tensorflow/models/blob/master/research/object_detection/g3doc/tf2.md

move the following files into the yolov5 folder: display.py, slice_os.py, stitch.py, frcnn_detect.py, overlap.py, inference_frcnn.py
Inference: run frcnn_detect.py --> possible arguments: '--source', root folder to image(s)
    --config, path to config
    --checkpt, path to checkpoint
    --labelmap, path to label map
    --output, path to output folder
    --conf-thres, confidence threshold
    --delete-image, dont save the image(only save bounding boxes)
